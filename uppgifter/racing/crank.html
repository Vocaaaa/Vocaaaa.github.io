Description

For ambitious tuning projects - 2Fast Crankshaft (47mm stroke / 95mm
connecting rod) Minarelli AM6 by renowned racing specialist 2Fast. This top-of-
the-line crankshaft, suitable for up to 20,000rpm, was developed to go with the
2Fast 100cc cylinder, but can be used with other cylinder kits as well.
Note: you'll have to do:

  • make room for the larger crank webs
  • make room for the connecting rod
  • use 20mm main bearings

Features:

  • stroke: 47mm
  • connecting rod: 95mm
  • small end bearing: 14 x 17 x 16.5mm
  • crankcase milling is necessary
